> Linux的重要目录和文件



| 颜色 |               文件类型               |
| :--: | :----------------------------------: |
| 蓝色 |                 目录                 |
| 绿色 |    可执行文件（脚本、二进制程序）    |
| 红色 |     压缩包（.tar, .zip, .gz 等）     |
| 青色 |       符号链接（类似快捷方式）       |
| 黄色 | 设备文件（/dev 下的块设备/字符设备） |
| 紫色 |     图片、音频、视频等多媒体文件     |
| 灰色 |       普通文件（文本、数据等）       |

# 系统核心与配置类

### `/boot`（启动文件：引导加载程序、内核、initrd）

```Bash
efi/                  # UEFI 模式下的 EFI 系统分区挂载点，存放引导文件
grub/                 # 引导程序的目录（配置、模块等）

vmlinuz-*             # 压缩的内核映像，启动时加载
initrd.img-*          # 初始 RAM 磁盘，挂载根文件系统前加载驱动和临时环境
System.map-*          # 内核符号表，用于调试
config-*              # 内核编译时的配置文件
```

### `/usr`（用户系统资源：应用程序、库、头文件、文档）

```Bash
bin/                  # 用户命令（普通用户可执行程序）
sbin/                 # 系统管理命令（通常需root权限）
lib/                  # 共享库及内核模块
libexec/              # 内部可执行文件（供其他程序调用）
include/              # C/C++头文件
share/                # 架构无关的共享数据（文档、字体、图标、man手册等）
local/                # 本地安装软件（手动编译或第三方软件）
src/                  # 源代码（如内核源码）

man                   # 手册页
```

### `/etc` （系统级配置文件：网络、服务、用户、sudo ）

```bash
# ========== 用户/组账户管理 ==========
shadow                # 用户密码加密信息（仅root可读）
passwd                # 用户账户信息（用户名/UID/GID/家目录/Shell）
group                 # 组账户信息
gshadow               # 组密码加密信息（仅root可读）
sudoers               # sudo权限主配置（必须用visudo编辑）
sudoers.d/            # sudo权限片段
# ========== 系统基础配置 ==========
hostname              # 系统主机名
hosts                 # 静态主机名到IP映射
localtime             # 系统时区
timezone              # 时区文本文件
fstab                 # 文件系统挂载表（开机自动挂载）
mtab                  # 当前已挂载的文件系统
nsswitch.conf         # 名称服务切换配置（hosts/passwd查询顺序）
sysctl.conf           # 内核参数配置（sysctl -p生效）
udev/                 # udev 设备管理规则
apt/                  # APT 包管理器配置（sources.list等）
lvm/                  # LVM逻辑卷管理配置目录
mdadm.conf            # 软件RAID管理配置
exports               # NFS共享导出配置
# ========== 网络配置与系统服务 ==========
resolv.conf           # DNS解析器配置（nameserver）
hosts.allow           # TCP Wrappers允许规则
hosts.deny            # TCP Wrappers拒绝规则
systemd/              # systemd系统和服务管理器配置目录
pam.d/                # PAM可插拔认证模块配置目录
security/             # 安全限制配置目录（limits.conf等）
ssh/                  # SSH服务端/客户端配置文件目录
sudoers.d/            # sudoers片段目录
# ========== 启动与初始化 ==========
default/grub          # GRUB引导加载器默认配置（生成/boot/grub/grub.cfg用）
skel/                 # 新用户家目录模板文件
profile               # 全局Shell初始化脚本（Bourne兼容）
bashrc                # 全局Bash初始化脚本
issue                 # 本地终端登录前显示信息
issue.net             # 远程登录前显示信息（如telnet）
motd                  # 登录后显示消息（通常为动态链接）
# ========== 计划任务 ==========
cron.d/               # 额外计划任务片段目录
cron.hourly/          # 每小时执行脚本目录
cron.daily/           # 每日执行脚本目录
cron.weekly/          # 每周执行脚本目录
cron.monthly/         # 每月执行脚本目录
crontab               # 系统级计划任务主配置
# ========== 日志管理 ==========
logrotate.d/          # 日志轮转片段目录
logrotate.conf        # 日志轮转主配置
```

# 用户与动态数据类

### `/var` （日志、状态数据、队列、缓存、临时文件、锁）

```Bash
# ========== 日志管理 ==========
log/                  # 系统及服务日志主目录
log/messages          # 系统通用日志（RHEL/CentOS）
log/syslog            # 系统通用日志（Debian/Ubuntu）
log/auth.log          # 认证日志（Debian/Ubuntu）
log/secure            # 认证日志（RHEL/CentOS）
log/kern.log          # 内核消息日志
log/boot.log          # 系统启动服务日志
log/dmesg             # 内核启动日志
log/maillog           # 邮件服务日志
log/cron              # 计划任务日志
log/lastlog           # 最近登录记录（二进制，lastlog读取）
log/wtmp              # 登录历史记录（二进制，last读取）
log/btmp              # 失败登录记录（二进制，lastb读取）
log/faillog           # 用户登录失败计数
log/journal/          # systemd日志持久化目录
log/nginx/            # Nginx访问/错误日志子目录
log/apt/              # APT包管理器日志
log/yum.log           # YUM包管理器日志
# ========== 运行时状态数据 ==========
lib/                  # 运行时状态数据主目录
lib/misc/             # 杂项状态（如locate数据库）
lib/systemd/          # systemd运行时状态（随机种子等）
lib/dhcp/             # DHCP客户端租约文件
lib/mlocate/          # locate数据库（mlocate.db）
opt/                  # 第三方软件运行时数据
# ========== 队列数据 ==========
spool/                # 队列数据主目录
spool/cron/           # cron任务文件（用户crontab）
spool/mail/           # 用户邮件存放目录（常链接到/var/mail）
spool/lpd/            # 打印队列（遗留）
# ========== 缓存数据 ==========
cache/                # 缓存主目录
cache/apt/archives/   # APT下载的deb包缓存
cache/yum/            # YUM缓存
cache/dnf/            # DNF缓存
cache/ldconfig/       # 动态链接器缓存
# ========== 临时文件 ==========
tmp/                  # 临时文件（重启清空，与/tmp类似）
tmp/systemd-private-* # systemd服务私有临时目录沙盒
# ========== 锁文件 ==========
lock/                 # 锁文件主目录（常链接到/run/lock）
lock/subsys/          # SysV服务锁文件目录
```

### `/proc`（进程与内核信息）

```Bash
[0-9]*/               # 每个进程的子目录（以PID命名，如1/，1234/）
self/                 # 指向当前进程的目录（如/proc/self）
acpi/                 # ACPI电源管理信息（温度、电池等）
bus/                  # 总线设备信息
fs/                   # 文件系统相关信息
irq/                  # 中断请求信息
net/                  # 网络协议栈信息（tcp、udp、dev等）
scsi/                 # SCSI设备信息
sys/                  # 内核参数可调（类似sysctl，可读写）
sysvipc/              # System V IPC资源（消息队列、共享内存、信号量）

cmdline               # 内核启动命令行参数
cpuinfo               # CPU型号、核心数、频率等
meminfo               # 内存使用详情（总内存、空闲、缓存、交换）
loadavg               # 系统平均负载（1/5/15分钟）
uptime                # 系统运行时间和空闲时间
version               # 内核版本和编译信息
filesystems           # 当前内核支持的文件系统列表
mounts                # 当前挂载的文件系统（等同于/etc/mtab）
partitions            # 系统识别的分区信息（主次设备号、大小）
swaps                 # 交换分区/文件使用情况
stat                  # 系统整体统计（中断、上下文切换、进程创建等）
interrupts            # 中断请求统计（CPU每核中断数）
ioports               # I/O端口使用情况
iomem                 # 物理内存映射（硬件设备占用的内存区域）
modules               # 已加载的内核模块（等同于lsmod）
diskstats             # 磁盘I/O统计（读写次数、扇区、时间）
buddyinfo             # 内存伙伴系统分配信息（排查内存碎片）
slabinfo              # 内核slab分配器信息（缓存使用）
zoneinfo              # 内存区域详细信息（DMA、Normal等）
sysrq-trigger         # 触发SysRq魔术键（调试/紧急恢复，需root）
kmsg                  # 内核日志消息（dmesg数据源，只读）
cgroups               # 控制组信息
schedstat             # 调度器统计
```

### `/home `（普通用户的家目录）

### `/root`（root 用户的家目录）

```Bash
.ssh/                 # SSH密钥及配置（authorized_keys、id_rsa、config）
.cache/               # 用户级缓存目录（浏览器缓存、pip缓存等）
.config/              # 用户级应用程序配置（systemd、git、htop等）
.local/               # 用户级本地数据（bin、share等）
.pip/                 # pip包管理器缓存/配置（Python）
.docker/              # Docker客户端配置（需安装Docker）
.npm/                 # Node.js npm缓存（需安装Node）
.vim/                 # Vim插件/备份目录
.aws/                 # AWS CLI配置（云服务）
.gnupg/               # GPG密钥环目录

.bashrc               # Bash初始化脚本（别名、环境变量）
.bash_history         # 命令历史记录（敏感操作审计）
.profile              # 通用Shell初始化（某些发行版使用）
.bash_profile         # 登录Shell配置（通常调用.bashrc）
.bash_logout          # 退出登录Shell时执行
.zshrc                # Zsh配置文件（如使用Zsh）
.vimrc                # Vim编辑器配置（缩进、颜色等）
.gitconfig            # Git全局配置（用户名、邮箱）
.wgetrc               # wget配置文件（代理、镜像）
.curlrc               # curl配置文件
.ssh/authorized_keys  # 免密登录公钥列表（重要安全）
.ssh/id_rsa           # SSH私钥（权限600，严禁泄露）
.ssh/id_rsa.pub       # SSH公钥
.ssh/config           # SSH客户端配置（跳板机、别名）
```

### `/run`（运行时变量数据：进程 PID 文件、系统服务状态，重启即清空）

```Bash
user/                 # 用户运行时目录（/run/user/<UID>）
systemd/              # systemd运行时状态、socket及通知路径
lock/                 # 全局锁文件目录（如 LCK.. 文件）
dbus/                 # D-Bus 系统总线 socket 目录（bus 文件）
NetworkManager/       # NetworkManager 运行时状态（连接、设备等）
mount/                # 辅助挂载点（如 /run/mount/utab）
log/                  # 部分服务运行时日志（journald 可能不直接在此）
udev/                 # udev 运行时数据（control socket、queue等）
tmpfiles.d/           # 临时文件配置片段（通常为 .conf 文件）

utmp                  # 当前登录用户记录（二进制，who/uptime 读取）
reboot-required       # Ubuntu 系统重启标志（内核更新后存在）
agetty.reload         # agetty 重新加载标志（某些发行版）
initctl               # 旧式 SysV init 控制管道（systemd 下通常没有）
sshd.pid              # SSH 守护进程 PID 文件（也可能在 /run/sshd.pid）
crond.reboot          # cron 记录系统重启时间的标记
xtables.lock          # iptables 操作互斥锁
```

### `/srv`（服务对外开放的数据: Web、FTP、版本控制）

```Bash
www/                  # Web服务数据（如Nginx/Apache站点根目录）
http/                 # Apache HTTP Server数据（部分发行版）
nginx/                # Nginx数据（站点配置或根目录）
ftp/                  # FTP服务数据（vsftpd等）
git/                  # Git仓库存储目录（Gitolite、GitLab）
svn/                  # Subversion版本库目录
samba/                # Samba共享数据目录（较少用，通常位于/var/lib/samba）
nfs/                  # NFS导出挂载点（有时位于/srv/nfs）
rsync/                # Rsync模块数据目录
postgresql/           # PostgreSQL数据库数据（有时在/var/lib/pgsql）
mysql/                # MySQL数据目录（通常位于/var/lib/mysql）
media/                # 多媒体服务数据（如DLNA、流媒体）
backup/               # 备份数据存放（按需）
```

# 虚拟与设备类

### `/dev`（设备文件：磁盘、终端、随机数等）

```Bash
shm/                  # 共享内存（tmpfs，用于进程间通信）
mqueue/               # POSIX消息队列（挂载点）
net/                  # 网络设备相关（如tun、tap）
disk/                 # 磁盘标识目录（by-id、by-path、by-uuid等，便捷管理磁盘）
input/                # 输入设备（鼠标、键盘等事件）
pts/                  # 伪终端从设备（SSH、终端模拟器用）
bus/                  # 总线设备（USB、PCI等）

null                  # 数据黑洞（丢弃所有写入）
zero                  # 无限零字节流
random                # 真随机数（阻塞，需熵池）
urandom               # 伪随机数（非阻塞，常用）
full                  # 模拟设备无空间错误（测试用）
sda                   # 第一块磁盘（完整磁盘）
sda1                  # 第一块磁盘第一分区
sdb                   # 第二块磁盘
nvme0n1               # 第一块NVMe固态硬盘（完整磁盘）
nvme0n1p1             # NVMe第一块盘第一分区
loop0                 # 环回设备0（挂载镜像文件）
tty                   # 当前控制终端
tty1                  # 第一个虚拟控制台（Alt+F1）
console               # 系统控制台（内核消息）
```

### `/sys`（内核对象、设备与驱动模型）

```Bash
block/                # 块设备目录（sda、nvme0n1等）
class/net/            # 网络接口类（eth0、ens33、wlan0等）
class/thermal/        # 热温控设备（温度传感器、冷却设备）
class/scsi_device/    # SCSI设备（磁盘扫描触发）
class/drm/            # 显卡DRM设备（GPU信息）
devices/              # 设备树（按总线、物理位置组织）
module/               # 已加载内核模块（可查看/修改模块参数）
kernel/debug/         # 内核调试接口（需挂载debugfs）
kernel/security/      # 安全相关接口（如LSM）
kernel/slab/          # slab分配器信息（类似/proc/slabinfo）
firmware/             # 固件相关接口
fs/                   # 文件系统特定参数（如cgroup、btrfs）
power/                # 电源管理（休眠状态、唤醒等）

scheduler             # I/O调度器（位于 block/sda/queue/）
stat                  # 块设备统计（位于 block/sda/）
operstate             # 网卡状态（up/down）（位于 class/net/eth0/）
speed                 # 网卡速率（Mbps）（位于 class/net/eth0/）
address               # MAC地址（位于 class/net/eth0/）
temp                  # CPU温度（毫摄氏度）（位于 class/thermal/thermal_zone0/）
cur_state             # 冷却设备当前状态（位于 class/thermal/cooling_device0/）
scaling_cur_freq      # CPU当前频率（kHz）（位于 devices/system/cpu/cpu0/cpufreq/）
scaling_governor      # CPU调频策略（同上目录）
current_clocksource   # 当前时钟源（位于 devices/system/clocksource/clocksource0/）
uevent_helper         # uevent辅助程序路径（位于 kernel/）
lockdown              # 内核锁定状态（位于 kernel/security/）
state                 # 系统休眠状态（位于 power/）
wakeup_count          # 唤醒计数（位于 power/）
```

### `/media`（可移动媒体自动挂载点：U 盘、光驱）

```Bash
cdrom/                # 光盘/DVD挂载点（常见于物理机）
cdrom0/               # 第一个光驱挂载点（多光驱）
cdrom1/               # 第二个光驱挂载点
usb/                  # USB设备挂载点（某些发行版自动挂载）
usb0/                 # 第一个USB设备挂载点
usb1/                 # 第二个USB设备挂载点
disk/                 # 磁盘分区挂载点（通用）
floppy/               # 软驱挂载点（遗留）
dvd/                  # DVD挂载点
sdX/                  # 按设备名挂载（如 sdb1，自动生成）
用户名/                # 以用户名为目录的动态挂载点（某些桌面环境自动挂载）
```

```Bash
data/                 # 临时挂载数据分区（如 /dev/sdb1）
backup/               # 挂载备份存储设备
iso/                  # 挂载ISO镜像文件（如 mount -o loop）
usb/                  # 临时挂载U盘
nas/                  # 挂载NFS/SMB网络存储
test/                 # 测试挂载点
win/                  # 挂载Windows分区（如NTFS）
share/                # 通用共享挂载点
```
