# 系统核心与配置类

### `/boot`（启动文件：引导加载程序、内核、initrd）



```Bash
efi/                  # UEFI 模式下的 EFI 系统分区挂载点，存放引导文件
grub/                 # 引导程序的目录（配置、模块等）
vmlinuz-*             # 压缩的内核映像，启动时加载
initrd.img-*          # 初始 RAM 磁盘，挂载根文件系统前加载驱动和临时环境
System.map-*          # 内核符号表，用于调试
config-*              # 内核编译时的配置文件
```

### `/usr`（用户系统资源：应用程序、库、头文件、文档）

```Bash
bin/                  # 用户命令（普通用户可执行程序）
sbin/                 # 系统管理命令（通常需root权限）
lib/                  # 共享库及内核模块
libexec/              # 内部可执行文件（供其他程序调用）
include/              # C/C++头文件
share/                # 架构无关的共享数据（文档、字体、图标、man手册等）
local/                # 本地安装软件（手动编译或第三方软件）
src/                  # 源代码（如内核源码）
```

### `/etc` （系统级配置文件：网络、服务、用户、sudo ）

```bash
# ========== 用户/组账户管理 ==========
shadow                # 用户密码加密信息（仅root可读）
passwd                # 用户账户信息（用户名/UID/GID/家目录/Shell）
group                 # 组账户信息
gshadow               # 组密码加密信息（仅root可读）
sudoers               # sudo权限主配置（必须用visudo编辑）
sudoers.d/            # sudo权限片段
# ========== 系统基础配置 ==========
hostname              # 系统主机名
hosts                 # 静态主机名到IP映射
localtime             # 系统时区
timezone              # 时区文本文件
fstab                 # 文件系统挂载表（开机自动挂载）
mtab                  # 当前已挂载的文件系统
nsswitch.conf         # 名称服务切换配置（hosts/passwd查询顺序）
sysctl.conf           # 内核参数配置（sysctl -p生效）
udev/                 # udev 设备管理规则
apt/                  # APT 包管理器配置（sources.list等）
lvm/                  # LVM逻辑卷管理配置目录
mdadm.conf            # 软件RAID管理配置
exports               # NFS共享导出配置
# ========== 网络配置与系统服务 ==========
resolv.conf           # DNS解析器配置（nameserver）
hosts.allow           # TCP Wrappers允许规则
hosts.deny            # TCP Wrappers拒绝规则
systemd/              # systemd系统和服务管理器配置目录
pam.d/                # PAM可插拔认证模块配置目录
security/             # 安全限制配置目录（limits.conf等）
ssh/                  # SSH服务端/客户端配置文件目录
sudoers.d/            # sudoers片段目录
# ========== 启动与初始化 ==========
default/grub          # GRUB引导加载器默认配置（生成/boot/grub/grub.cfg用）
skel/                 # 新用户家目录模板文件
profile               # 全局Shell初始化脚本（Bourne兼容）
bashrc                # 全局Bash初始化脚本
issue                 # 本地终端登录前显示信息
issue.net             # 远程登录前显示信息（如telnet）
motd                  # 登录后显示消息（通常为动态链接）
# ========== 计划任务 ==========
cron.d/               # 额外计划任务片段目录
cron.hourly/          # 每小时执行脚本目录
cron.daily/           # 每日执行脚本目录
cron.weekly/          # 每周执行脚本目录
cron.monthly/         # 每月执行脚本目录
crontab               # 系统级计划任务主配置
# ========== 日志管理 ==========
logrotate.d/          # 日志轮转片段目录
logrotate.conf        # 日志轮转主配置
```

# 用户与动态数据类

### 数据顶层目录：/var （日志、缓存、邮件、锁文件）

### /tmp	临时文件（重启清空，权限 1777）

### /home	普通用户的家目录

### /root	root 用户的家目录

### /srv	服务提供的特定数据（如 Web 网站根目录）

### /media	可移动媒体自动挂载点（U 盘、光驱）

### /mnt	临时手动挂载点

```Bash
/var/log/	           # 所有日志文件目录
/var/log/audit/      # 审计日志（安全合规必查）
/var/spool/	         # 待处理任务队列（cron、邮件、打印）
/var/lib/	           # 数据库、容器、包管理器的持久化数据
/var/tmp/	           # 持久化临时文件（重启不清空）
/run/user/<uid>/	   # 每个登录用户的运行时目录（systemd 管理）
/var/log/syslog  # 系统日志
/var/log/nginx/  # Web 服务器日志
/var/cache/      # 缓存数据

```

# 虚拟与设备类

### /dev（设备文件：磁盘、终端、随机数等）

### /proc（进程与内核信息： /proc/cpuinfo, /proc/meminfo, /proc/pid）

### /sys（内核对象、设备与驱动模型）

### /run（运行时变量数据：进程 PID 文件、系统服务状态，重启即清空）













/boot	启动文件（内核 vmlinuz、初始内存盘 initrd、GRUB 配置）
/dev	设备文件（磁盘 /dev/sda、随机数 /dev/random、终端 /dev/tty）
/etc	系统配置文件（几乎所有的文本配置都在这）



/home	普通用户的家目录（/home/用户名）

/media	可移动媒体挂载点（U 盘、光盘自动挂载于此）
/mnt	临时挂载点（手动挂载外部文件系统）



/opt	可选应用程序软件包（第三方预编译软件，如 /opt/teamviewer）
/proc	虚拟文件系统，进程和内核信息（如 /proc/cpuinfo）
/root	root 用户的家目录（注意不在 /home 下）



/sys	虚拟文件系统，内核对象和设备/驱动信息（比 /proc 更结构化）
/tmp	临时文件（重启后通常清空，权限 1777）





/usr	用户系统资源（绝大多数应用程序、库、文档都在此）

/sbin	系统管理命令（如 fdisk, mount），通常链接到 /usr/sbin/

/bin	基础用户命令（如 ls, cp），通常链接到 /usr/bin

/lib	共享库和内核模块，通常链接到 /usr/lib





/var	可变数据（日志 /var/log、缓存 /var/cache、邮件 /var/mail、锁文件）

/run











/bin：基本用户命令，所有用户都能用（单用户模式也可用）
/sbin：系统管理命令，通常需要 root 权限
/usr/bin：大部分用户命令（系统启动后挂载）

/proc/cpuinfo    # CPU 信息
/proc/meminfo    # 内存信息
/proc/version    # 内核版本
/proc/uptime     # 系统运行时间
/proc/[PID]/     # 每个进程的信息目录

-- /dev设备
/dev/sda         # 第一块 SATA/SCSI 硬盘
/dev/sda1        # 第一块硬盘的第一个分区
/dev/nvme0n1     # NVMe 固态硬盘
/dev/tty         # 当前终端
/dev/random      # 随机数生成器
/dev/null        # 黑洞（丢弃所有写入）
