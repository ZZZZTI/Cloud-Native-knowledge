> 用户，组，权限

------

### 用户与组管理

```bash
# 查看用户信息
id [用户名]              # 查看用户UID、GID及所属组(系统用户:UID=1~999,普通用户:UID>=1000)
whoami                  # 显示当前用户名
who/w                   # 详细登录用户信息
last/lastlog            # 查看历史登录记录/用户最后登录时间
su [-] 用户名            # 切换用户（- 加载目标用户环境变量）
groups username         # 查看指定用户所属组
getent passwd 用户名     # 从数据库获取用户信息
getent group 组名        # 获取组信息

# 修改用户信息
useradd [选项] 用户名     # 创建用户
-u UID, -g/G 组名, -m(生成家目录), -s(shell), -r(系统用户), -c "xxx"(用户描述)
usermod [选项] 用户名     # 修改用户
-l new old (改名), -d dir/(改家目录), -g/-G group(改组)
passwd [选项] 用户名      # 设置/修改密码
-l (锁定), -u (解锁), -d (删除密码)
userdel [-r] 用户名      # 删除用户（-r 同时删除家目录和邮件池）

# 用户组
groupadd (-g 2000)(-r) groupname                 #创建组(指定GID)(创建系统组)
groupmod (-n new old)/(-g 2500) groupname        #修改组名/GID
groupdel groupname                               # 删除组（确保组内无用户）
gpasswd -M/a user1,user2 groupname               # 设置组成员列表
gpasswd -d user groupname                        # 从组中移除用户
gpasswd groupname                                # 设置组密码
chown [ -R(递归修改)] 所有者[:组] 文件/目录...       # 修改文件或目录的所有者和组
```

### 权限

```Bash
# 修改文件或目录的权限
chmod [ -R(递归修改)] 权限 文件/目录...
# 权限格式
u/g/o/a -> +/-/= -> r/w/x        # 符号表示
(421)-> 421-> 421-> 421          # 数字表示
4xxx(s)  # 执行时获得所有者权限
2xxx(s)  # 执行时获得所属组权限
1xxx(t)  # 在目录中只能删自己的文件
# 常用权限
     u   g   o
755（rwx r-x r-x）                # 脚本文件标准
644（rw- r-- r--）                # 配置文件标准
600（rw- --- ---）                # 私密文件（私钥）
700（rwx --- ---）                # 私有目录（家目录）
1777（rwx rwx rwt）               # /tmp 目录
4755（rws r-x r-x）               # passwd 命令（需要SUID修改 /etc/shadow）
2750（rwx r-s ---）               # 团队共享目录（SGID保证继承组）

# 文件权限第一位：类型
- 普通文件, d 目录, l 链接, b/c 设备, s 套接字, p 管道
```
