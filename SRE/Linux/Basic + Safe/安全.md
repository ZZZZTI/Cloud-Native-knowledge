> 防火墙堵口 ，SSH加固守门，日志留痕溯源

------

### 防火墙

```Shell
ufw enable/disable/reload      # 启用（开机自启）/ 禁用 / 重启
ufw status [verbose/numbered]  # 查看状态（默认策略、日志级别/带编号显示规则）
ufw reset                      # 重置（清除所有自定义规则，恢复初始禁用状态）
ufw app list                   # 查看所有已注册应用
ufw app info OpenSSH           # 查看应用详细信息

# 默认策略： 阻止/允许 外部发起连接/主动对外连接
ufw allow ssh -> ufw default allow/deny incoming/outgoing

# 管理访问规则
ufw allow/deny/limit [服务名] / [端口/协议] / [from IP(to any port 端口 proto tcp)]
#   放行/拒绝/限制速率    服务     服务端端口               特定IP访问特定端口

# 场景：Web服务器最小化开放端口
ufw default deny incoming
ufw default allow outgoing
ufw allow ssh                  # 管理端口
ufw allow http                 # 80
ufw allow https                # 443
ufw limit ssh                  # SSH增加限速
ufw enable

# 删除访问规则
ufw delete [allow 22/tcp] / 3

# 日志管理
ufw logging on/off              # 开启(low)/关闭日志
ufw logging low/medium/high     # 日志级别：记录被阻止的包/被阻止+允许的包/所有包
```

### SSH加固

```Shell
密钥登录配置流程

# 1.客户端生成密钥对：~/.ssh/id_rsa（私钥）和id_rsa.pub（公钥）
ssh-keygen -t rsa -b 4096    

# 2.把公钥传到服务器
ssh-copy-id -i ~/.ssh/id_rsa.pub user@服务器IP -p 2222

# 3.SSH安全加固（开多终端修改）
修改配置文件：/etc/ssh/sshd_config
PermitRootLogin no                    # 禁止root直接登录
PasswordAuthentication no             # 禁用密码登录（改用密钥）
PubkeyAuthentication yes              # 开启密钥认证
Port 2222                             # 更改默认22端口（防扫描）
MaxAuthTries 3                        # 最大重试次数
ClientAliveInterval 300               # 空闲超时自动断连
AllowUsers user1 user2                # 白名单用户（仅允许特定用户）
DenyUsers/Groups [user/group]         # 黑名单
  
sshd -t                               # 改完检查语法
systemctl restart sshd                # 改完执行
```

### 强制访问控制

```Shell
# AppArmor
aa-status                         # 查看所有已加载配置文件及其模式
aa-enforce /path/to/bin           # 强制模式：阻止违规操作
aa-complain /path/to/bin          # 投诉模式：仅记录日志
aa-genprof /path/to/bin           # 交互式生成新配置文件
aa-logprof                        # 根据日志事件更新现有配置文件
```

### 日志审计

```Shell
# 登录日志
last                        # 查看最近成功登录记录（含时间和IP）
lastb                       # 查看最近失败的登录尝试（暴力破解看这里）
lastlog                     # 查看所有用户最后一次登录时间

# 系统认证日志
/var/log/auth.log
/var/log/messages
/var/log/secure（认证）
/var/log/maillog

auditd（内核级审计）
systemctl start auditd
配置 /etc/audit/rules.d/audit.rules
auditd -w /etc/passwd -p wa -k identity             # 监控passwd写操作
auditd -w /etc/sudoers -p wa -k sudoers             # 监控sudoers变更
auditd -a always,exit -S unlink -S rmdir -k delete  # 监控删除行为
ausearch -k identity                                # 查看审计结果

AIDE（文件完整性检查）
aide --init
mv /var/lib/aide/aide.db.new.gz /var/lib/aide/aide.db.gz
aide --check            # 定期对比（可配合cron做巡检）
```
